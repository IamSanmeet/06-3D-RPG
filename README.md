# Project 06 RPG
Last project for MSCH-C220
04/29/2023

A simple RPG with a score, targets, an enemy, and dialogue.

## Implementation
Godot Engine

Features:  
Flying Enemy  
NPC Dialogue  
Fluid Movement through a 3D space  
Shoot and Targets  
Health (Enemy) and Score  
Win and Lose screen
